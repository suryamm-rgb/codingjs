# 📅 March --- First Week Coding Plan (Basic Level)

## 👨‍💻 Level

Beginner → Beginner+

## 🎯 Goal

Build strong programming basics using: - Loops - Conditions - Number
Logic - Strings - Arrays - Patterns

## 💻 Recommended Language

JavaScript

# 🟢 Monday --- March 2 (Number Logic -- Level 2)

1.  Check Leap Year\
    Input: 2024\
    Output: Leap Year

2.  Count Even & Odd Digits\
    Input: 12345\
    Output: Even: 2, Odd: 3

    Input: 10\
